# Email Automation Package

A Python-based email automation tool for sending bulk emails efficiently.

## Features

- Bulk email sending capability
- Email template support
- SMTP configuration
- CSV contact import
- Customizable email subjects and body

## Installation

```bash
pip install -r requirements.txt
```

## Usage

1. Configure your SMTP settings in the config file
2. Add your email contacts to contacts.csv
3. Run the main script:
```bash
python main.py
```

## Configuration

Edit the configuration file to set:
- SMTP server and port
- Email credentials
- Default subject and body
